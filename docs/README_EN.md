## Description

  

Web application for managing film collection with rating system, categories and customizable themes.

  

**Main features:**

- Film and category management
- 10-point rating system
- Search and filtering with Kinopoisk data loading
- Flexible theme system
- Data and category export/import



  

---

  

## Installation and Launch

  

```bash

# Clone repository

git clone https://github.com/FLEXIY0/KinoTeka-watchlist.git

cd KinoTeka-watchlist

  

# Install dependencies

npm install

  

# Launch

npm start

```

  

**Windows:** run `start_app.bat or start.bat`  

**Linux/macOS:** run `./start.sh`

  

Application will open at: `http://localhost:3000`
