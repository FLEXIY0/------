## Описание

  

Веб-приложение для управления коллекцией фильмов с системой рейтингов, категорий и настраиваемыми темами оформления. 

  

**Основные функции:**

- Управление фильмами и категориями
- Рейтинговая система (10-бальная)
- Поиск и фильтрация с подгрузкой с Кинопоиска
- Гибкая система тем
- Экспорт/импорт данных и категорий


  

---

  

## Установка и запуск

  

```bash

# Клонировать репозиторий

git clone https://github.com/FLEXIY0/KinoTeka-watchlist.git

cd KinoTeka-watchlist

  

# Установить зависимости

npm install

  

# Запустить

npm start

```

  

**Windows:** запустите `start_app.bat` или `start.bat`  

**Linux/macOS:** запустите `./start.sh`

  

Приложение откроется по адресу: `http://localhost:3000`
